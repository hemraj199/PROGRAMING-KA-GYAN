CSS

Definition

CSS (Cascading Style Sheets) is used to style and layout webpages, including changes in color, font, spacing, and positioning.

Common CSS Properties with Examples

  • color: Sets the text color. Example: color: red;
  • font-size: Defines text size. Example: font-size: 20px;
  • margin: Adds space around elements. Example: margin: 10px;
  • padding: Adds space inside elements. Example: padding: 15px;
  • border: Adds a border. Example: border: 2px solid black;

History of CSS

CSS was introduced in 1996 by the W3C to separate content from design. Over time, CSS evolved into multiple versions, with CSS3 introducing advanced features like animations and grid layouts.
