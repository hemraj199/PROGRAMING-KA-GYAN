HTML

Definition

HTML (HyperText Markup Language) is the standard language used to create and design webpages.

Common HTML Tags with Examples

  • <p>: Defines a paragraph. Example: <p>This is a paragraph</p>
  • <h1>-<h6>: Defines headings. Example: <h1>Heading 1</h1>
  • <a>: Defines a hyperlink. Example: <a href="https://www.example.com">Visit Example</a>
  • <img>: Embeds an image. Example: <img src="image.jpg" alt="Example Image">
  • <table>: Creates a table. Example: <table><tr><td>Cell</td></tr></table>

History of HTML

HTML was developed by Tim Berners-Lee in 1991. Since then, it has evolved through multiple versions, with HTML5 being the latest standard.
