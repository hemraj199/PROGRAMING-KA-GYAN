JavaScript

Definition

JavaScript is a versatile programming language used to create dynamic and interactive content on websites.

Common JavaScript Methods with Examples

  • document.getElementById: Selects an element by ID. Example: document.getElementById('demo')
  • console.log: Prints output to the console. Example: console.log('Hello World');
  • alert: Displays a popup message. Example: alert('Welcome!');
  • addEventListener: Attaches an event handler. Example: element.addEventListener('click', function() {});

History of JavaScript

JavaScript was created by Brendan Eich in 1995 and has since evolved into one of the most widely used programming languages for web development.
